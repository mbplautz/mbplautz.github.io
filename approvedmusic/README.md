# APlayer Playlist Tool

This is a client-side only web app for playing a custom list of MP3 files with the APlayer framework.

## Features

- Paste a playlist in the form `Track Name|https://example.com/song.mp3`
- Drag in local MP3 files from your browser
- Load a demo playlist instantly
- Save the playlist in browser storage so it persists across refreshes

## Run it

Open [index.html](index.html) directly in a browser, or serve the folder with a simple static server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.
